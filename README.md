## Installation

## Running Server

```bash
  git clone https://github.com/rohanshrestha09/Patra-Nestjs.git
  cd Patra-Nestjs
  npm install
  npm run start:dev
```

## Running Client

```bash
  git clone https://github.com/rohanshrestha09/Patra-Nestjs.git
  cd Patra-Nestjs/client
  npm install
  npm run start
```
